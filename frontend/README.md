# Frontend

Implementation of IDecide project frontend is built with React.js

## How to run
1. ```cd frontend```
2. ```cd idecide```
3. ```yarn```
4. ```npm start```

    Local:            http://localhost:3000
  On Your Network:  http://192.168.0.8:3000
